# Configuration Files
Configuration file and setup script for my Ubuntu setup.

## Get Started
```sh
bash install.sh
```
## Tmux Shortcuts
```
# install plugins
ctrl + j then I (shift i)

# create window
ctrl + j & c

# window management
ctrl + j & [window number]
ctrl + j & p (previous window)
ctrl + j & n (next window)

# scroll text
ctrl + [ & j or k
```

## Neovim Shortcuts 
```
"<leader>sk": see all keymaps (leader key is space)
```
